## Context
We need a lightweight way to record architectural decisions so future contributors understand why choices were made.

## Decision
Adopt ADR (Architecture Decision Records) using the template in `docs/adr/template.md`. Each decision gets a numbered markdown file.

## Consequences
- All significant decisions are documented and searchable.
- New contributors can understand historical context.
